Compile:
- Run 'make' on the CLI on any unix based system with cmake and it will create the bin/pmt, if you run 'make path', and copy paste the output in the CLI you can use the tool by calling it directly ie: 'pmt'
Running: 
- For more info on running, compile and run 'pmt -h'
